Bienvenue sur le dépôt de distribution officiel de **SmartAquaWiz**. Ce dossier contient uniquement les fichiers de production prêts à l'emploi.

## 🚀 Téléchargements Directs

* **[Télécharger l'application Windows (AquaWiz.zip)](./AquaWiz.zip?raw=true)** : L'archive contenant l'application complète. Décompresse simplement le dossier et lance `AquaWiz.exe` !
* **[Télécharger le Firmware ESP32 (firmware.bin)](./firmware.bin?raw=true)** : Le binaire d'usine à flasher sur ton microcontrôleur.

## ⚡ Instructions d'installation rapide

1. **Pour l'ESP32** : Utilise l'outil d'installation intégré dans l'application Windows pour injecter le fichier `firmware.bin` ou passe par l'interface de mise à jour Cloud (OTA) directement depuis le portail Wi-Fi de ton appareil.
2. **Pour Windows** : Lance simplement `AquaWiz.exe` (exécute-le en mode Administrateur si nécessaire pour la détection des ports COM).
